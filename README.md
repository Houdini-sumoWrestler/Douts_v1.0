# DOUTS Club Management System

## Overview

This project is a custom club management system being designed for DOUTS. The goal of the system is to provide a more flexible and dive-club-specific alternative to existing club management software.

The system is intended to support:

* Member registration and authentication
* Dive register and certification management
* Event creation and registration
* Organiser approval workflows
* Club asset and gear booking
* Payment processing
* Access logging and audit trails
* Meeting and committee management
* Email notifications

The project is currently in the planning and architecture stage.
